# AI HealthGuard - DESIGN.md

**UI / UX DESIGN REQUIREMENTS**
*   **Theme:** Professional health-tech website look, not a basic dashboard.
*   **Aesthetic:** Clean, modern, premium UI similar to top healthcare SaaS products.
*   **Color Palette:** Green & teal gradient background (health-themed, calming).
*   **Typography:** High contrast text with fully visible fonts (no faded or low-opacity text). Use **Arial MT** or similar clean sans-serif font.
*   **Components:** Rounded cards, soft shadows, glassmorphism where appropriate.
*   **Layout:** Smooth spacing and alignment for a desktop-first layout.

**TECH & STRUCTURE EXPECTATIONS**
*   UI should be modular, scalable, and developer-ready.
*   Layout must clearly support integration with Flask / Streamlit backend.
*   Design should work smoothly for hackathon demo and real-world deployment.
