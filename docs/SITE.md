# AI HealthGuard - SITE.md

## 1. Vision
AI HealthGuard is a modern AI-powered healthcare decision-support web application. It allows users to input symptoms and patient details to receive AI-powered health risk assessments, emergency predictions, insurance guidance, and nearby hospital recommendations. The UI must be professional, reassuring, and highly intuitive.

## 2. Core Features
*   **Symptom Selection:** Intuitive search and categorization of 50+ symptoms.
*   **Patient Details Intake:** Easy input for age, symptom severity, duration, and city.
*   **Analysis Dashboard:** Clear presentation of risk scores, predicted conditions, and influential symptoms.
*   **Insurance Guidance:** Eligibility checks and actionable claim advice.
*   **Hospital Recommendations:** Location-based hospital finding with navigation.

## 3. Technology Stack
*   **Frontend:** React (Mocked for now via Stitch generation)
*   **Styling:** Custom CSS based on DESIGN.md
*   **Backend Integration:** Designed to easily integrate with Flask/Streamlit APIs.
